# QuantumHZ - Meditation App 🧘‍♀️

QuantumHZ is a modern and intuitive meditation application specially designed for iOS users. It is developed to support mental peace and wellness.

## 📋 About the Project

QuantumHZ is an iOS application designed to help meditation practitioners integrate mindfulness into their daily routines. The app offers various meditation sessions and guided exercises to help users reduce stress and find mental peace.

### Key Features
- 🎧 Guided meditation sessions
- ⏱️ Customizable meditation durations
- 📊 User progress tracking
- 🎨 User-friendly and modern interface
- 📱 Fully optimized for iOS devices

## 🛠️ Technology Stack

- **Language**: Swift
- **Platform**: iOS
- **Development Environment**: Xcode
- **Project Type**: Native iOS Application

## 📁 Project Structure

```
QuantumHZ/
├── QuantumHz.xcodeproj/        # Xcode project files
├── QuantumHz/                  # Main application source code
├── Views/                      # SwiftUI view components
├── Meditations/                # Meditation content and data
├── LICANCE                      # License file
└── README.md                    # This file
```

### Directory Descriptions

- **QuantumHz.xcodeproj/**: Xcode project configuration files used by Xcode. Build settings, targets, and dependencies are defined here.

- **QuantumHz/**: Main source code directory of the application. Models, ViewControllers, and other core components are located here.

- **Views/**: View components written using SwiftUI. UI elements such as the meditation interface, home screen, and settings page are located here.

- **Meditations/**: Contains meditation session content, audio files, and meditation metadata.

## 🚀 Installation and Running

### Requirements
- macOS 12.0 or later
- Xcode 14.0 or later
- iOS 15.0 or later target device

### Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/fikoture/QuantumHZ.git
   cd QuantumHZ
   ```

2. **Open in Xcode**:
   ```bash
   open QuantumHz.xcodeproj
   ```

3. **Configure build settings**:
   - Select the `QuantumHz` target in Xcode
   - Set up your Apple Developer Account in the Signing & Capabilities section

4. **Run the app**:
   - Select a simulator or physical iOS device
   - Click the ▶️ Play button or press `Cmd + R`

## 📖 Usage

1. **Start the App**: View meditation options on the home screen
2. **Select a Meditation**: Choose your desired meditation session
3. **Practice**: Follow the guided meditation
4. **Track Progress**: View completed sessions from your profile

## 🏗️ Development

### Code Architecture

The project uses SwiftUI and Model-View-ViewModel (MVVM) architecture:

- **Models**: Meditation data, user sessions, and progress information
- **ViewModels**: UI logic and data binding
- **Views**: SwiftUI components and interface elements
